# Oil DGA App - Starter

این پروژه یک اپ ساده HTML/CSS/JS برای تحلیل اولیه تست روغن ترانسفورماتور است.

## اجرا
۱. پوشه را در VS Code باز کنید.
۲. افزونه Live Server را نصب کنید.
۳. روی index.html راست‌کلیک کنید و Open with Live Server را بزنید.

## فایل‌ها
- index.html: ساختار صفحه
- style.css: ظاهر صفحه
- data.js: ۲۰ نمونه پایان‌نامه و نتایج مرجع
- diagnosisEngine.js: هسته تشخیص و قوانین
- app.js: اتصال فرم و دکمه‌ها به موتور تشخیص
